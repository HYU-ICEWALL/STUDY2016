Day2: 조건과 반복

이제 우리는 프로그램을 좀 더 효율적으로 작성하는 방법을 배울거에요.

프로그램은 작성될 당시에 우리가 예측할 수 있는 모든 경우를 생각해야해요.

또한 그에 따른 각각의 행동도 미리 정의해두어야 하죠.

프로그램이 입력된 값에 따라서 분기를 나누게 되는데 이를 if를 사용하여 만들 수 있어요.

if는 조건이 참일때만 그 구문을 실행해주는 역활을 해요.

if (조건)
{
/*codes*/
}

if.c

위의 예제를 통해서 if를 사용하는 다양한 방법들을 익힐 수 있어요.

조건과 더불어 같은 코드를 여러번 반복하고 싶을때는 반복문을 사용할 수 있어요.

while이 대표적인 반복문인데, if와 매우 비슷해요

while (조건)
{
/*codes*/
}

while.c

반복문은 조건이 충족될때 동안 계속 반복하기 때문에 조건이 맞지않을 경우를 만들지 않으면 무한정 반복해요.

보통 예제에 나온 a같은 변수를 반복문 내에서 증감하여 반복문을 탈출할 수 있도록 만들어요.

그러다보니 귀찮은 프로그래머들은 어차피 이런 a같은 변수를 만들어야 될 꺼면 좀 더 간단하게 쓰고자 for라는걸 만들었어요.

for(선언; 조건; 반복)
{
/*codes*/
}
