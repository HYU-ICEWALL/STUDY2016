# 1번: 문자열 처리

이름과 생년월일, 각종 개인정보를 입력받아서 자기소개를 하는 프로그램을 출력해봅시다.

**입력값:**<br>
>배지운<br>
>950623<br>

**출력값:**<br>
>이름은 배지운이고<br>
>95년 06월 23일 생입니다.<br>

굳이 같은 형식일 필요는 없으며 다양한 정보를 추가로 입력받아서 출력해보아요.<br>
